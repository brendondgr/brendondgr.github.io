A web scraping and data management application to discover, organize, and refine fellowship opportunities.

GitHub Google API Perplexity API

// About The Project

Fellowship Finder automates the tedious process of searching for professional fellowships. It began by targeting ProFellow.com and is expanding to include private resources, job boards, and other major platforms. The core mission is to provide a centralized, efficient tool for users to discover and manage opportunities with the help of AI-powered data refinement. It utilizes either Google Gemini API or Perplexity API to refine the data, to get the most accurate information.

// Key Features

Interactive Web App (`app.py`)

A Flask-powered web interface to view, search, filter, and manage opportunities. Favorite the best ones and hide the irrelevant ones for a personalized view.

Scraper & AI Refiner (`data_retrieval.py`)

The core script uses Selenium to scrape fellowship data and then leverages Google's Generative AI to clean, standardize, and enhance it into a structured dataset.

Automated Scheduler (`driver.py`)

This script acts as a scheduler, automatically running the data retrieval process at set intervals. Set it and forget it, and always have the latest data.

// Command-Line Interface

app.py

Runs the Flask web server for the user interface. It has no command-line flags.

python app.py

data_retrieval.py

The main script for scraping, processing, and cleaning data.

Flag Description Example Command
(no flag) Scrape for new data using the browser specified in `configs/filters.json`. 
python data_retrieval.py
--refine Process raw data with the AI refiner without scraping. 
python data_retrieval.py --refine
--cleartmp Clear the temporary `tmp/` folder. 
python data_retrieval.py --cleartmp
--cleardata Clear the `data/` folder. 
python data_retrieval.py --cleardata
--cleanup Clear both `tmp/` and `data/` folders for a fresh start. 
python data_retrieval.py --cleanup

driver.py

A scheduler to run the data retrieval script automatically.

Flag Description Example Command
(no flag) Run the scheduler continuously, triggering data retrieval every 2 hours by default. 
python driver.py
--interval N Set the interval between runs to N hours. 
python driver.py --interval 4
--run-once Run the data retrieval script just once and then exit. 
python driver.py --run-once

// Getting Started

Follow these steps to get a local copy up and running.

1. Clone the Repository

First, clone the repository to your local machine.

git clone https://github.com/brendondgr/Fellowship-Finder.git
cd Fellowship-Finder

2. Set Up Credentials

Set up your API keys as environment variables. For Gemini, use "GEMINI_API_KEY" or "GOOGLE_API_KEY". For Perplexity, use "PPLX_API_KEY" or "PERPLEXITY_API_KEY". The application defaults to using Perplexity if available.

export GEMINI_API_KEY="YOUR_GEMINI_API_KEY_HERE"
export PPLX_API_KEY="YOUR_PERPLEXITY_API_KEY_HERE"

Also, add your ProFellow.com login details to configs/login.json.

3. Install & Run (uv)

Install dependencies and run the application using uv (recommended).

# Install dependencies
uv sync

# Run the application
uv run app.py
